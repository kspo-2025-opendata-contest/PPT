# 숨인재 발표자료

2025 국민체육진흥공단 공공데이터 활용 경진대회 대상(1위) 발표자료를 웹으로 구현한 프로젝트입니다.

## 기술 스택

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Recharts

## 실행 방법

**Prerequisites:** Node.js

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 개발 서버 실행:
   ```bash
   npm run dev
   ```
